# 🚀 AI Content Generation Engine

An intelligent AI-powered content generation system built using Flask.  
This project enables dynamic generation of multiple content types such as LinkedIn posts, professional emails, and advertisements using advanced prompt engineering techniques.

---

## 📌 Project Overview

The AI Content Generation Engine is designed to create personalized, structured, and high-quality content based on user-defined parameters such as:

- Tone (Professional, Casual, Persuasive, Friendly)
- Target Audience
- Content Length (Short, Medium, Long)
- Keyword Inclusion
- Content Type Selection

The system uses modular architecture for scalability and maintainability.
